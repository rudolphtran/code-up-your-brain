4. Variables

üí° Concept:

Variables are "containers" that store data in memory. Each variable has:

  • Name
  • Value

Example:

name = "Minh"
age = 12
print("My name is", name)
print("I am", age, "years old")

‚öôÔ∏è Variable Naming Rules:

  • Only contains letters, numbers, and underscore _
  • Cannot start with a number
  • Case sensitive

‚úÖ Valid examples:

student_name = "An"
age2 = 13

‚ùå Invalid examples:

2name = "An"      # Error: starts with number
student-name = 12 # Error: contains dash
