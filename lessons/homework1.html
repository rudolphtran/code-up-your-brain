Exercise 1: Personal Information

Exercise 2: Rectangle Perimeter and Area

Input length and width of a rectangle. Calculate and print: Perimeter, Area

Input bill amount and tip percentage. Calculate and print: Tip amount, Total amount to pay

Exercise 5: Calculate BMI

Input weight (kg) and height (m). Calculate BMI using formula: BMI = weight / (height^2)

Important Notes

  1. Use meaningful variable names: length instead of a
  2. Comment your code: Explain what you're doing
  3. Test multiple cases: Try with negative numbers, zero, decimals
  4. Format output nicely: Use f-strings for readable results
  5. Basic error handling: Check for valid input
