1. Introduction to Conditionals

üéØ Learning Objectives:

  • Understand what conditional statements are and their importance
  • Learn to think logically about program flow
  • Recognize real-world scenarios that require decision-making
  • Prepare for writing your first conditional code

üß† What are Conditional Statements?

Conditional statements are the "decision-makers" of programming. They allow your code to choose different paths based on specific conditions, just like how you make decisions in real life.

üè† Real-Life Examples:

  • Weather: "If it's raining, take an umbrella"
  • Traffic: "If the light is red, stop the car"
  • Shopping: "If I have enough money, buy the item"
  • Study: "If I finish homework, I can play games"

üíª Programming Thinking:

1
Identify the Condition

"What needs to be checked?"

2
Define the Actions

"What happens if true? What happens if false?"

3
Write the Logic

"Translate the decision into code"

üëÄ Your First Look at Conditional Code:

# Simple decision making
age = 18

if age >= 18:
    print("You can vote!")
else:
    print("You cannot vote yet.")
Output: 
You can vote!

Don't worry if this looks confusing now - we'll learn each part step by step!

2. Comparison Operators

üéØ Learning Objectives:

  • Master all 6 comparison operators in Python
  • Understand the difference between = and ==
  • Practice writing comparison expressions
  • Learn common mistakes and how to avoid them

üîç What are Comparison Operators?

Comparison operators are the tools we use to compare values. They always return either True or False - these are called Boolean values.

üìä Complete Comparison Operators Table:

Operator Name Example Result Explanation
== Equal to 5 == 5 True Checks if two values are exactly the same
!= Not equal to 5 != 3 True Checks if two values are different
> Greater than 8 > 3 True Left value is larger than right value
< Less than 2 < 7 True Left value is smaller than right value
>= Greater than or equal 5 >= 5 True Left value is larger than or equal to right value
<= Less than or equal 3 <= 10 True Left value is smaller than or equal to right value

‚ö†Ô∏è Critical Distinction:

= (Assignment)

Stores a value in a variable

age = 25  # Storing 25 in age variable
VS
== (Comparison)

Checks if two values are equal

age == 25  # Returns True or False

üß™ Interactive Examples:

Example 1: Numeric Comparisons
# Let's test some numbers
a = 10
b = 20
c = 10

print(f"a = {a}, b = {b}, c = {c}")
print(f"a == c: {a == c}")      # True (same value)
print(f"a != b: {a != b}")      # True (different values)
print(f"b > a: {b > a}")        # True (20 > 10)
print(f"a < b: {a < b}")        # True (10 < 20)
print(f"a >= c: {a >= c}")      # True (10 >= 10)
print(f"a <= b: {a <= b}")      # True (10 <= 20)
Output: 
a = 10, b = 20, c = 10
a == c: True
a != b: True
b > a: True
a < b: True
a >= c: True
a <= b: True
Example 2: String Comparisons
# Strings can also be compared
name1 = "Alice"
name2 = "Bob"
name3 = "Alice"

print(f"name1 == name3: {name1 == name3}")  # True
print(f"name1 != name2: {name1 != name2}")  # True
print(f"name1 > name2: {name1 > name2}")    # False (alphabetical order)
print(f"name2 > name1: {name2 > name1}")    # True
Output: 
name1 == name3: True
name1 != name2: True
name1 > name2: False
name2 > name1: True

3. Basic if Statements

The if statement is the most basic conditional statement. It executes code only when a condition is true.

üìù Syntax:

if condition:
    # Code to execute if condition is True
    statement1
    statement2

‚ö†Ô∏è Important:

  • Don't forget the colon (:) after the condition
  • Indent the code inside the if block (4 spaces or 1 tab)

Example:

# Simple if statement
temperature = 30

if temperature > 25:
    print("It's hot today!")
    print("Consider wearing light clothes.")

print("Have a nice day!")  # This always runs
Output: 
It's hot today!
Consider wearing light clothes.
Have a nice day!

4. if-else and elif Statements

if-else Statements

When you want to do something different if the condition is false, use else.

üìù Syntax:

if condition:
    # Code when condition is True
else:
    # Code when condition is False

Example:

# Check if a number is positive or negative
number = -5

if number > 0:
    print(f"{number} is positive")
else:
    print(f"{number} is negative or zero")
Output: 
-5 is negative or zero

elif Statements

When you have multiple conditions to check, use elif (else if).

üìù Syntax:

if condition1:
    # Code for condition1
elif condition2:
    # Code for condition2
elif condition3:
    # Code for condition3
else:
    # Code when all conditions are False

Example - Grade Calculator:

# Grade calculator
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Your grade is: {grade}")
Output: 
Your grade is: B

üí° Tip:

Python checks conditions from top to bottom and stops at the first true condition.

5. Logical Operators

Logical operators allow you to combine multiple conditions.

Operator Meaning Example
and Both conditions must be true age >= 18 and age <= 65
or At least one condition must be true day == "Saturday" or day == "Sunday"
not Reverses the condition not is_raining

Example - Voting Eligibility:

# Check voting eligibility
age = 20
is_citizen = True

if age >= 18 and is_citizen:
    print("You can vote!")
elif age >= 18 and not is_citizen:
    print("You must be a citizen to vote.")
else:
    print("You must be 18 or older to vote.")
Output: 
You can vote!

6. Project: Leap Year Checker

üí° Project Goal:

Create a program that determines if a given year is a leap year using conditional statements.

üìã Leap Year Rules:

1

If a year is divisible by 4, it might be a leap year

2

If it's also divisible by 100, it's NOT a leap year

3

UNLESS it's also divisible by 400, then it IS a leap year

‚úÖ Examples:

2024

‚úÖ Leap year

Divisible by 4, not by 100
1900

‚ùå Not leap year

Divisible by 4 and 100, but not 400
2000

‚úÖ Leap year

Divisible by 4, 100, and 400
2023

‚ùå Not leap year

Not divisible by 4

üî® Step-by-step Solution:

Step 1: Basic Structure
# Get input from user
year = int(input("Enter a year: "))

# We'll add the logic here
print(f"Checking if {year} is a leap year...")
Step 2: Apply the Rules (Nested Approach)
# Leap year checker - nested approach
year = int(input("Enter a year: "))

if year % 4 == 0:
    if year % 100 == 0:
        if year % 400 == 0:
            print(f"{year} is a leap year")
        else:
            print(f"{year} is not a leap year")
    else:
        print(f"{year} is a leap year")
else:
    print(f"{year} is not a leap year")
Step 3: Optimized Version (Logical Operators)
# More concise leap year checker
year = int(input("Enter a year: "))

if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year} is a leap year! üéâ")
    print("February will have 29 days this year.")
else:
    print(f"{year} is not a leap year.")
    print("February will have 28 days this year.")

7. Project: Electricity Bill Calculator

üí° Project Goal:

Create a tiered electricity billing system based on usage levels.

üí∞ Pricing Structure:

Usage Range (kWh) Price per Unit Description
0 - 100 $0.10 Basic tier - lowest rate
101 - 200 $0.15 Standard tier - medium rate
201+ $0.20 Premium tier - highest rate

üìä Example Calculation:

Usage: 250 kWh

Tier 1: 100 kWh √ó $0.10 = $10.00
Tier 2: 100 kWh √ó $0.15 = $15.00
Tier 3: 50 kWh √ó $0.20 = $10.00
Total Bill: $35.00

üî® Step-by-step Solution:

Step 1: Get Input and Validate
# Get electricity usage from user
units = float(input("Enter electricity usage (kWh): "))

# Validate input
if units < 0:
    print("‚ùå Invalid input! Usage cannot be negative.")
else:
    print(f"Calculating bill for {units} kWh...")
Step 2: Calculate Using Conditions
# Electricity bill calculator
units = float(input("Enter electricity usage (kWh): "))

if units < 0:
    print("‚ùå Invalid input! Usage cannot be negative.")
else:
    bill = 0
    
    if units <= 100:
        # All units in first tier
        bill = units * 0.10
    elif units <= 200:
        # First 100 units at tier 1, rest at tier 2
        bill = (100 * 0.10) + ((units - 100) * 0.15)
    else:
        # All three tiers
        bill = (100 * 0.10) + (100 * 0.15) + ((units - 200) * 0.20)
    
    print(f"üí° Your electricity bill: ${bill:.2f}")
Step 3: Enhanced Version with Breakdown
# Enhanced electricity bill calculator with detailed breakdown
units = float(input("Enter electricity usage (kWh): "))

if units < 0:
    print("‚ùå Invalid input! Usage cannot be negative.")
else:
    bill = 0
    print(f"\nüìä Bill Breakdown for {units} kWh:")
    print("-" * 40)
    
    if units <= 100:
        bill = units * 0.10
        print(f"Tier 1 ({units:.1f} kWh @ $0.10): ${bill:.2f}")
    elif units <= 200:
        tier1 = 100 * 0.10
        tier2 = (units - 100) * 0.15
        bill = tier1 + tier2
        print(f"Tier 1 (100.0 kWh @ $0.10): ${tier1:.2f}")
        print(f"Tier 2 ({units - 100:.1f} kWh @ $0.15): ${tier2:.2f}")
    else:
        tier1 = 100 * 0.10
        tier2 = 100 * 0.15
        tier3 = (units - 200) * 0.20
        bill = tier1 + tier2 + tier3
        print(f"Tier 1 (100.0 kWh @ $0.10): ${tier1:.2f}")
        print(f"Tier 2 (100.0 kWh @ $0.15): ${tier2:.2f}")
        print(f"Tier 3 ({units - 200:.1f} kWh @ $0.20): ${tier3:.2f}")
    
    print("-" * 40)
    print(f"üí° Total Bill: ${bill:.2f}")
    
    # Add usage tip
    if units > 200:
        print("üí° Tip: Consider reducing usage to save money!")
    elif units > 100:
        print("üí° Good usage level!")
    else:
        print("üí° Excellent! You're using energy efficiently!")
