Exercise 1: Personal Information

Write a program that asks the user to input: Full name, Age, City

Then print to screen in format:

Hello [Name], you are [Age] years old, from [City]

Exercise 2: Rectangle Perimeter and Area

Input length and width of a rectangle. Calculate and print: Perimeter, Area

Exercise 3: Convert Celsius to Fahrenheit

Input temperature in Celsius, convert to Fahrenheit and print result.

Formula: F = C * 9/5 + 32

Exercise 4: Calculate Tip

Input bill amount and tip percentage. Calculate and print: Tip amount, Total amount to pay

Exercise 5: Calculate BMI

Input weight (kg) and height (m). Calculate BMI using formula: BMI = weight / (height^2)

Intermediate Exercises (6-10)

Exercise 6: Simple Calculator

Input 2 numbers and 1 operator (+, -, *, /, %, **). Print the result of that calculation.

Exercise 7: Time Conversion

Input number of seconds, convert and print in format: [hours] hours [minutes] minutes [seconds] seconds

Exercise 8: Calculate Salary

Input: Hours worked, Hourly wage. Calculate total salary (if > 40 hours, calculate overtime x1.5)

Exercise 9: Divide Candy

Have N candies, divide equally among M people. Calculate: How many candies each person gets, How many candies are left over

Exercise 10: Calculate Exact Age

Input birth year, calculate and print: Current age, Number of days lived (approximate)

Important Notes

  1. Use meaningful variable names: length instead of a
  2. Comment your code: Explain what you're doing
  3. Test multiple cases: Try with negative numbers, zero, decimals
  4. Format output nicely: Use f-strings for readable results
  5. Basic error handling: Check for valid input

Goals
